Quick FAQ

Data Privacy: How can I delete my data?

• The /score command is used to manage your saved score for the commands, select "reset" to clear your points (you cannot restore it after this action!!!) and "unhide" if you've hidden your score.

• If you're using a custom prefix and not the deafult "?"" in your server that will also save your server ID in the DB, to clear it simply change the prefix back to "?"

• If you enabled /reply-to-dms and it's set to true, your user ID is also stored to know who to reply to, to clear it set the option to false.
